# ParallelMatrixProduct-
ParallelMatrixProduct is the program for matrix product.

## Installation

```bash
~/$ ./start
```

## Usage
```bash
~/$ cd bin
~/bin/java src.Main
```

After those steps follow commands which appear in command line.
